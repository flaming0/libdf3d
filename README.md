libdf3d
=======
Simple C++ game framework aimed for mobile platforms.

#### Currently supported
* desktop platforms: windows, linux, mac os x
* android (partially)
* windows phone (partially)

#### Compiler
* The project heavily uses c++11 and c++14 features.
